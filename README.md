# E-Commerce Sales and Customer Analysis

## Project Overview
This project analyzes an e-commerce retail dataset to understand sales performance, customer behavior, product trends, and profitability patterns.

The aim of the analysis is to answer key business questions around:
- sales growth over time
- regional performance
- customer contribution
- product performance
- the relationship between discounting and profit

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Tableau

## Files in this Repository
- `notebooks/` → Jupyter Notebook analysis
- `data/` → cleaned dataset used for reporting and dashboarding
- `dashboard/` → dashboard screenshots
- `report/` → exported report files

## Key Insights
- Sales performance varies across regions, with some regions contributing significantly more revenue.
- A relatively small number of customers account for a large share of total sales.
- Higher discount levels are often associated with lower profitability.
- Product and category performance vary substantially, highlighting opportunities for targeted strategy.

## Project Deliverables
- Data cleaning and preparation
- Exploratory data analysis
- Business insight generation
- Visual reporting
- Dashboard-ready cleaned dataset
